Bibliotecas Estándar de C y C++

Bibliotecas Estándar de C

Las bibliotecas estándar de C proporcionan una amplia gama de funciones para manejar operaciones básicas como entrada/salida, manipulación de cadenas, y manejo de memoria. A continuación se enumeran algunas de las bibliotecas más comunes:

<stdio.h>

Proporciona funciones para la entrada y salida estándar, como printf y scanf.

                
                    #include <stdio.h>
            
                    int main() {
                        printf("Hola, Mundo!\n");
                        return 0;
                    }

<stdlib.h>

Incluye funciones para manejo de memoria, control de procesos, y conversiones de cadenas, como malloc, free, y atoi.

                
                    #include <stdlib.h>
            
                    int main() {
                        int *p = (int *)malloc(sizeof(int));
                        *p = 10;
                        printf("Valor: %d\n", *p);
                        free(p);
                        return 0;
                    }

<string.h>

Proporciona funciones para manipulación de cadenas, como strlen, strcpy, y strcmp.

                
                    #include <string.h>
            
                    int main() {
                        char str1[] = "Hola";
                        char str2[] = "Mundo";
                        strcat(str1, str2);
                        printf("Cadena concatenada: %s\n", str1);
                        return 0;
                    }

<math.h>

Incluye funciones matemáticas como sqrt, pow, y sin.

                
                    #include <math.h>
            
                    int main() {
                        double raiz = sqrt(16.0);
                        printf("Raíz cuadrada de 16: %f\n", raiz);
                        return 0;
                    }

<time.h>

Proporciona funciones para manipulación del tiempo y la fecha, como time y localtime.

                
                    #include <time.h>
            
                    int main() {
                        time_t ahora = time(NULL);
                        struct tm *tm_info = localtime(&ahora);
                        printf("Fecha y hora actual: %s", asctime(tm_info));
                        return 0;
                    }

Bibliotecas Estándar de C++

Las bibliotecas estándar de C++ extienden las funcionalidades de C y añaden características orientadas a objetos, manejo de cadenas, colecciones, y mucho más. Algunas de las bibliotecas estándar más importantes son:

<iostream>

Proporciona soporte para entrada y salida mediante objetos como std::cin, std::cout, y std::cerr.

                
                    #include <iostream>
            
                    int main() {
                        std::cout << "Hola, Mundo!" << std::endl;
                        return 0;
                    }

<string>

Incluye la clase std::string para manipulación de cadenas de caracteres.

                
                    #include <string>
            
                    int main() {
                        std::string saludo = "Hola";
                        saludo += " Mundo";
                        std::cout << saludo << std::endl;
                        return 0;
                    }

<

Proporciona la clase std::vector, una estructura de datos dinámica similar a un array.

                
                    #include <vector>
            
                    int main() {
                        std::vector numeros = {1, 2, 3, 4, 5};
                        for (int num : numeros) {
                            std::cout << num << " ";
                        }
                        std::cout << std::endl;
                        return 0;
                    }

<

Incluye funciones algoritmicas como sort, find, y copy.

                
                    #include <algorithm>
                    #include <vector>
            
                    int main() {
                        std::vector numeros = {5, 3, 1, 4, 2};
                        std::sort(numeros.begin(), numeros.end());
                        for (int num : numeros) {
                            std::cout << num << " ";
                        }
                        std::cout << std::endl;
                        return 0;
                    }

<map>

Proporciona la clase std::map, una estructura de datos asociativa basada en árboles balanceados.

                
                    #include <map>
            
                    int main() {
                        std::map edades;
                        edades["Alice"] = 30;
                        edades["Bob"] = 25;
                        for (const auto &par : edades) {
                            std::cout << par.first << ": " << par.second << std::endl;
                        }
                        return 0;
                    }
