Funciones y Sobrecarga de Operadores en C++

Funciones

Las funciones en C++ permiten encapsular un bloque de código que realiza una tarea específica. Pueden aceptar parámetros y devolver un valor.

Definición y Declaración

                
                    // Declaración de una función
                    int suma(int a, int b);
            
                    // Definición de la función
                    int suma(int a, int b) {
                        return a + b;
                    }
            
                    int main() {
                        int resultado = suma(5, 3);
                        std::cout << "Resultado: " << resultado << std::endl;
                        return 0;
                    }

En el ejemplo anterior, la función suma toma dos enteros como parámetros y devuelve su suma.

Funciones Sobrecargadas

La sobrecarga de funciones permite definir varias funciones con el mismo nombre pero con diferentes parámetros.

                
                    // Sobrecarga de funciones
                    int suma(int a, int b) {
                        return a + b;
                    }
            
                    double suma(double a, double b) {
                        return a + b;
                    }
            
                    int main() {
                        int resultadoInt = suma(5, 3);
                        double resultadoDouble = suma(5.5, 3.3);
                        std::cout << "Resultado entero: " << resultadoInt << std::endl;
                        std::cout << "Resultado double: " << resultadoDouble << std::endl;
                        return 0;
                    }

En este ejemplo, la función suma está sobrecargada para aceptar tanto enteros como dobles.

Sobrecarga de Operadores

La sobrecarga de operadores permite redefinir el comportamiento de los operadores para tipos de datos definidos por el usuario.

Definición de Operadores Sobrecargados

Para sobrecargar un operador, se define una función miembro o una función amiga que usa el operador como nombre.

                
                    #include <iostream>
            
                    class Vector {
                    private:
                        int x, y;
                    public:
                        Vector(int x, int y) : x(x), y(y) {}
            
                        // Sobrecarga del operador +
                        Vector operator+(const Vector &v) const {
                            return Vector(x + v.x, y + v.y);
                        }
            
                        void mostrar() const {
                            std::cout << "(" << x << ", " << y << ")" << std::endl;
                        }
                    };
            
                    int main() {
                        Vector v1(1, 2);
                        Vector v2(3, 4);
                        Vector v3 = v1 + v2; // Usa el operador + sobrecargado
                        v3.mostrar();
                        return 0;
                    }

En este ejemplo, el operador + está sobrecargado para sumar dos objetos de la clase Vector.

Operadores Miembros y Amigo

Los operadores pueden ser definidos como funciones miembros (como en el ejemplo anterior) o como funciones amigas para acceder a los datos privados.

                
                    class Vector {
                    private:
                        int x, y;
                    public:
                        Vector(int x, int y) : x(x), y(y) {}
            
                        // Sobrecarga del operador << como función amiga
                        friend std::ostream& operator<<(std::ostream &os, const Vector &v) {
                            os << "(" << v.x << ", " << v.y << ")";
                            return os;
                        }
                    };
            
                    int main() {
                        Vector v(1, 2);
                        std::cout << v << std::endl; // Usa el operador << sobrecargado
                        return 0;
                    }

En este ejemplo, el operador << está sobrecargado como una función amiga para permitir la impresión de objetos Vector usando std::cout.
