Clases y Objetos en C++

En C++, la programación orientada a objetos (POO) se basa en la creación y manipulación de clases y objetos. Las clases son plantillas para crear objetos, y los objetos son instancias de clases.

Definición de una Clase

Una clase define un tipo de dato complejo que puede contener datos (miembros de datos) y funciones (métodos) que operan sobre esos datos.

                
                    class MiClase {
                    private:
                        int dato; // Miembro de datos privado
            
                    public:
                        // Constructor
                        MiClase(int valor) : dato(valor) {}
            
                        // Método público
                        void mostrarDato() {
                            std::cout << "Dato: " << dato << std::endl;
                        }
            
                        // Método para modificar el dato
                        void setDato(int valor) {
                            dato = valor;
                        }
                    };

Creación de Objetos

Un objeto es una instancia de una clase. Se puede crear un objeto utilizando la sintaxis del constructor de la clase.

                
                    int main() {
                        // Crear un objeto de MiClase
                        MiClase obj(10);
            
                        // Usar los métodos del objeto
                        obj.mostrarDato(); // Muestra: Dato: 10
                        obj.setDato(20);
                        obj.mostrarDato(); // Muestra: Dato: 20
            
                        return 0;
                    }

Encapsulamiento

El encapsulamiento es un principio de la POO que restringe el acceso a algunos de los datos de un objeto y los protege de modificaciones externas no deseadas. Esto se logra usando especificadores de acceso como private, protected, y public.

                
                    class MiClase {
                    private:
                        int dato; // Miembro de datos privado
            
                    public:
                        // Métodos públicos para acceder y modificar dato
                        void setDato(int valor) {
                            dato = valor;
                        }
            
                        int getDato() {
                            return dato;
                        }
                    };

Herencia

La herencia permite que una clase herede características (miembros y métodos) de otra clase. Esto facilita la reutilización del código y la creación de jerarquías de clases.

                
                    class Base {
                    public:
                        void mostrar() {
                            std::cout << "Clase Base" << std::endl;
                        }
                    };
            
                    class Derivada : public Base {
                    public:
                        void mostrar() {
                            std::cout << "Clase Derivada" << std::endl;
                        }
                    };
            
                    int main() {
                        Derivada obj;
                        obj.mostrar(); // Muestra: Clase Derivada
            
                        return 0;
                    }

Polimorfismo

El polimorfismo permite que las funciones se comporten de manera diferente según el tipo de objeto que las llama. Esto se logra utilizando funciones virtuales y sobrecarga de operadores.

                
                    class Base {
                    public:
                        virtual void mostrar() {
                            std::cout << "Clase Base" << std::endl;
                        }
                    };
            
                    class Derivada : public Base {
                    public:
                        void mostrar() override {
                            std::cout << "Clase Derivada" << std::endl;
                        }
                    };
            
                    int main() {
                        Base *ptr = new Derivada();
                        ptr->mostrar(); // Muestra: Clase Derivada
            
                        delete ptr;
                        return 0;
                    }
