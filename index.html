Estructuras Básicas de Datos

Arrays: Descripción: Colección de elementos del mismo tipo, accesibles por índices. Ejemplo 

>> int arr[5] = {1, 2, 3, 4, 5};

Estructuras (Structs)

Estructuras (Structs): Descripción: Agrupación de variables de diferentes tipos bajo un mismo nombre. Ejemplo: 

struct Persona {
    char nombre[50];
    int edad;
};

istas Enlazadas

Listas Enlazadas: Descripción: Estructura donde cada elemento apunta al siguiente. Ejemplo: 

                    >> int arr[5] = {1, 2, 3, 4, 5};

Pilas (Stacks) y Colas (Queues):

Descripción: Pilas siguen el principio LIFO (Last In First Out) y colas el FIFO (First In First Out). Ejemplo de pila

                    #define MAX 100 int pila[MAX]; int top = -1; void push(int x) { if (top == MAX - 1) return; // Overflow pila[++top] = x; } int pop() { if (top == -1) return -1; // Underflow return pila[top--]; }
